Comandos Gerais:

  1. !corno [membro] - Gera uma porcentagem de “corno”.
  2. !gado [membro] - Gera uma porcentagem de “gado”.
  3. !amor [membro] - Gera uma porcentagem de “amor”.
  4. !piroca [membro] - Gera o tamanho da “piroca” em cm.

Comandos de Interação:

  1. !tinder - Escolhe uma pessoa aleatória do chat.
  2. !eusou - Gera um adjetivo aleatório para você.
  3. !eununca - Pergunta se você nunca fez algo.
  4. !pergunta - Faz uma pergunta para você.

Comandos DBD (Dead by Daylight):

  1. !chase - Escolhe uma pessoa aleatória do chat.
  2. !santuario - Mostra o santuário de perks atual do DBD.
  3. !passe - Exibe quantos dias faltam para o passe do DBD acabar.
  4. !caiu [numero] - Comando para usar quando você caiu numa chase do DBD. O parâmetro de número não precisa ser especificado, ele sempre incrementa +1.
  5. !rcaiu [numero] - Mesmo estilo do !caiu, só que decrementa o valor de “caiu”. O parâmetro de número não precisa ser especificado, sendo assim, sempre vai decrementar -1.
  6. !morreu [numero] - Comando para usar quando você morreu numa partida do DBD. O parâmetro de número não precisa ser especificado, sendo assim, sempre vai incrementar +1.
  7. !rmorreu [numero] - Mesmo estilo do !morreu, só que decrementa o valor de “morreu”. O parâmetro de número não precisa ser especificado, sendo assim, sempre vai decrementar -1.

Comandos de Streaming:

  1. !followage [membro] - Exibe há quanto tempo tal pessoa segue o canal do streamer.
  2. !uptime - Exibe há quanto tempo o atual streamer está ao vivo.

Comando de Entretenimento:

  1. !onlyfans - Fake Onlyfans.

Comandos para Streamer / Moderadores:

  1. !s2
  2. !modulo [nome] [status] - Desativa ou ativa comandos / eventos. Por exemplo, !modulo corno 0 desativa o comando !corno. Para ativar, use 1 em vez de 0.
